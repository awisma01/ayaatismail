# Personal Website

This is a personal website created to showcase the work and research of me as a PhD student and CHI Fellow at Michigan State University. The website includes sections for research interests, current projects, writing, and contact information.

## Project Structure

```
personal-website
├── assets
│   └── styles
│       └── style.css      # CSS styles for the website
├── index.html              # Main HTML document for the website
└── README.md               # Documentation for the project
```

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd personal-website
   ```

3. **Open the `index.html` file** in your web browser to view the website.

## Features

- Responsive design that adapts to different screen sizes.
- Smooth scrolling navigation for easy access to different sections.
- Sections for research interests, current projects, writing, and contact information.

## Technologies Used

- HTML5
- CSS3
- JavaScript (for smooth scrolling functionality)
